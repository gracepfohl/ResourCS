We Offer


Web Development

Web Development is simply creating websites for the World Wide Web.

We have four levels of Web Development that increase in difficulty, and you can choose the level you want to start with based on your prior knowledge. This section is for you if you are looking to make basic, interactive, and visually appealing websites and web apps.

Video Game Development

Game Development is exactly as it sounds: building games.

We have two levels to choose from based on your experience level. If you want to learn basic computer science logic and how to use Python, this section is for you.

Web App Development

Web Application Development is the creation of software applications that can be accessed from the web.

We offer two levels that allow you to create your own web applications using Django/Python as well as Ruby on Rails/Ruby.
