How To Stay Organized


Before coding a website or program, you computer needs to know what language you’re writing in; otherwise, the page will not load. For example, if your document is in HTML, you need to save the file as *yourtitle.html*. It is important to title your files correctly in order to let your text editor and computer know what language your code is written in.

Our Guide

Here is how you save files in common languages:


HTML: *title*.html

CSS: *title*.css

PYTHON: *title*.py

JAVA: *title*.java

JAVASCRIPT: *title*.js

TEXT FILE: *title*.txt

C SHARP: *title*.cs

SQL: *title*.sql

C++: *title*.cpp

PHP: *title*.php

RUBY/RAILS: *title*.rb

Note that you should not use spaces anywhere in your file names. Doing so will only create problems when referencing your files later.

In addition, it is important to save your files in the correct directory(folder). If you’re working on a project and you’re hosting it on GitHub, you should stay organized and know where all your folders are located.
